# Synopsis

This app permit to save a batch of products in category and generate an URL to share with contacts. Built with React Native.

## Functionalities

- Save a category name.
- Save product description.
- Generate URL address.
- Share URL address

## Directory Structure

```bash

eshopMobileClient

│   ├── CategoryScreen.js
│   ├── ProductScreen.js
├── App.js
├── app.json
├── index.js
├── package-lock.json
├── package.json
└── README.md

```

## Running the project

Clone this repository :

```bash
git clone https://github.com/ChrisBapin/eshop-mobile-client.git

cd eshopMobileClient
```

Install packages

```bash
npm install
```

Start the server:

```bash
npm start
```
